# Installation!
Install gem dependencies
```
bundle install
```
Create Database
```
rails db:setup
```
Install frontend dependencies (install yarn: https://yarnpkg.com/en/docs/install, node 6+: https://nodejs.org/en/)
```
yarn
```

Run with
```
foreman start
```

Test with
```
rspec
```
**Note:** Requires `PhantomJS` installed
